# ApplaudoTest

ApplaudoTest is a Cypress io Framework that has 4 automated test cases for a technical interview.


## Usage

# install brew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# install node.js
brew install node

```cypress
npm install cypress

# run the tests
npm run cy:run
